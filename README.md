## Test automation framework for Store Cpanel Application

### Prerequisites:

- Node.js version 20+
- npm version 8+

### Tools:

1. Node v20.10.0
2. Typescript
3. Playwright
4. ESlint
5. Prettier

### Getting started:

#### Install project dependencies

```bash
npm install
```

or

```bash
npm ci
```

### Run tests

```bash
npm run test
```

or

```bash
npm t
```

### HTML Report

```bash
npm run report
```
